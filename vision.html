Vision

When values are exchanged, it is not the physical value that change hands. It is the information attached to the value that is swapped

If openBarter is organized as a central limit order book, a depositories (green) can use several markets (red), forming a web of decentralized exchange. The centrality of the order book just means that competition occurs on orders it contains.
