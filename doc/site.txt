Le repertoire principal du site est $OBREP = ./site
les scripts apache serveur apache pointent sur $OBREP/w
les utilitaires sont dans $OBREP/x
les medias pointent $OBREP/m
les fichiers de données pointent sur $OBDATA = $OBREP/d
le fichier de config $OBREP/x/ob.conf
les logs pointent sur $OBREP/l

Automate:
c'est un script $OBREP/x/automate.py lançé par cron.
il ne se lance pas quand le fichier $OBREP/d/proc est présent
il crée $OBREP/d/proc et écrit le numéro de processus dedans
il fait son travail
quand c'est terminé il efface le fichier $OBREP/d/proc
